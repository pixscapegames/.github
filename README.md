## What is Pixscape?

Pixscape is a growing ecosystem for 2D game creation and runtime tooling.

It is built around a simple idea:

> powerful 2D workflows should stay readable, fast, and pleasant to use.

The organization currently focuses on:

- **Pixscape Studio** — the editor
- **Pixscape Runtime** — the runtime foundation

---

## Core projects

### [`pixscape-studio`](https://github.com/pixscapegames/pixscape-studio)
The visual editor for building 2D scenes and game content.

### [`pixscape-runtime`](https://github.com/pixscapegames/pixscape-runtime)
The runtime layer powering rendering, tiled content, animation, particles, physics, and more.

---

## Philosophy

Pixscape is designed around a few strong principles:

- **Fast iteration**
- **Clear structure**
- **Lightweight workflows**
- **Modern 2D tooling**
- **A strong balance between usability and power**

The goal is not to create a bloated toolchain, but a focused ecosystem that stays efficient and pleasant over time.

---

## Current focus

The project is actively evolving, with ongoing work around:

- tiled workflows
- parallax and layered rendering
- animation and particles
- physics integration
- asset management
- editor usability
- future intelligent tooling for content creation
